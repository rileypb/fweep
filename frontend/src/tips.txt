fweep is meant to be used with your keyboard, though it will work with the mouse too.
Pressing / will focus the keyboard on the game prompt.
Entering \ before a command marks it as a map-making command.
Using double quotes around a room name (e.g., "back room") will differentiate it from otherwise ambiguous rooms (e.g., "laundromat back room").
Press R, then click empty canvas to place a new room at the pointer location.
Press N, then click empty canvas to place a new sticky note at the pointer location.
Drag from a room's directional handle to create a connection. Drop on empty space to create a new target room automatically.
Use the Up and Down Arrow keys in the game input to walk through your command history.
Commands like \create Kitchen, \connect Kitchen to Hallway, \edit Kitchen, \show Kitchen, and \delete Kitchen all work directly from the game input.
Bare directions like \north and \south can navigate from the currently selected room when that exit exists.
Use go to <room name> in the game to jump selection and center the map on a known room.
Type \notate <room name> to drop a sticky note linked to a room without leaving the keyboard.
The game accepts \arrange and \prettify as layout commands to tidy the current map automatically.
Type \zoom in, \zoom out, or \zoom reset in the game input to control the map zoom without leaving the keyboard.
Type \zoom 25, \zoom 200, or \zoom 200% in the game input to jump straight to a specific zoom level between 25% and 300%.
Use Alt+Shift+M to jump back to the map list.
Use Alt+Shift+G to toggle the background grid.
Use Alt+Shift+S to toggle grid snapping.
Use Alt+Shift+C to switch between curved and straight connection styles.
Use Alt+Shift+V to toggle the map's visual style.
Use Alt+Shift+Y to cycle through the canvas themes.
Use Alt+Shift+D to switch between light and dark app themes.
Use Alt+Shift+P to prettify the current layout.
Use Alt+Shift+O to open the background image controls.
Use Alt+Shift+J to export the current map as JSON.
Use Alt+Shift+E to export the current map as a PNG image.
Use Alt+Shift+R to reset the linked game panel when a game is running.
Use Alt+Shift+F to open a story file from your device when no game is running yet.
Press L to toggle the selected room's lock state.
Use the arrow keys on the focused map canvas to move selection from room to room.
Press Enter on the focused map canvas to open the selected room editor.
Press Delete or Backspace on the focused map canvas to delete the current selection.
Press +, -, and 0 on the focused map canvas to zoom in, zoom out, or reset zoom.
Double-click a room or connection to open its room editor.
Shift-click rooms or connections to add them to the current selection.
Use the Parchment panel to search IFDB, load a local story file, or play a linked game beside the map.
The background image popup lets you import a reference image, adjust its zoom, and recenter it with Option-drag or Command-drag on the canvas.
The minimap overview helps you keep large maps navigable and shows where your current viewport sits.
Export PNG supports entire-map, selection, and region capture modes.
The map name chip also shows the linked game's title when a story is associated with the map.
The help dialog includes a keyboard-shortcuts section if you want the full cheat sheet later.
